Getting started

Installation#

You can install the plugin via WP Admin, WP CLI, or by downloading it from WordPress.org.

Via WP Admin:#

  1. Go to WP Admin -> Plugins -> Add New
  2. Search for "Servebolt Optimizer"
  3. Find the plugin in the results and click "Install Now"
  4. After installation is done click "Activate"

Via WP CLI:#

Run the following command to download and activate the plugin:

wp plugin install servebolt-optimizer --activate

Download from WordPress.org:#

  1. Navigate to https://wordpress.org/plugins/servebolt-optimizer/
  2. Download the Zip-file by clicking "Download"
  3. Go to WP Admin -> Plugins -> Add New -> Upload Plugin
  4. Select the Zip-file and click "Install Now"
  5. After upload click "Activate Plugin"
